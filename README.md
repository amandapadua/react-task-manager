Gerenciador de Tarefas
Aplicação web de gerenciamento de tarefas desenvolvida com React e Vite, criada como projeto de aprendizado durante o curso de React para Iniciantes.
🖥️ Demonstração

Adicione, visualize, complete e delete tarefas com persistência local e dados externos via API.

✨ Funcionalidades

✅ Adicionar tarefas com título e descrição
✅ Marcar tarefas como concluídas
✅ Deletar tarefas
✅ Ver detalhes de cada tarefa em página separada
✅ Persistência de dados com localStorage
✅ Consumo de API externa (JSONPlaceholder)
✅ Navegação entre páginas com React Router

🚀 Tecnologias

React
Vite
React Router DOM
Tailwind CSS v4
Lucide React
JSONPlaceholder API
